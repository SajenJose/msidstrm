Project Description:

This project introduces a utility created in C# which helps to understand some of the less understood features of the Windows Installer database. The utility helps to learn the various data steams embedded within the Windows Installer database.

Features of the MSI Data Stream Utility:

  • Read and list data streams embedded within the Windows Installer database
  • Add various data streams into the Windows Installer database
  • Delete data streams from the Windows Installer database
  • Extract data streams from the Windows Installer database
  • Extract files embedded within cab files inside the Windows Installer database
  • Converts GUID to Compressed / Packed / Squished Format and vice versa.
  • List, Create and Extract files from a Cab File
  • Lists the files & folder path lengths from a provided input path (recursively)
